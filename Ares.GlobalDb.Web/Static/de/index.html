Was ist RPGMusicTags.de?

RPGMusicTags.de ist eine Datenbasis mit Musik-Tags für (pen & paper)-Rollenspiele. Das heißt, hier sind für verschiedene Musikstücke Tags gespeichert, welche diese Musik beschreiben. Tags sind in Kategorien eingeteilt und können z.B. eine Stimmung ("spannend", "traurig", ...), einen Ort ("Dungeon", "Orient", ...) oder eine Situation ("Kampf", "Reise", ...) beschreiben.

Wie kann ich die Daten nutzen?

Am einfachsten geht das mit einem Programm, welche für die Musik, die man bereits auf dem Rechner hat, die Tags aus der Datenbasis holt. Derzeit kann das der Musik- und Soundplayer Ares. Im Player kann man dann direkt Tags auswählen, die zur aktuellen Situation während des Spiels passen, und es wird entsprechende Musik abgespielt.

Um sich Inspiration zu holen, um zu sehen, was eigentlich schon in der Datenbasis drin ist, oder um passende Musik überhaupt erst zu finden, kann man die Datenbasis auch hier online durchstöbern.

Woher kommen die Daten?

Von Leuten, die für Musikstücke bereits Tags festgelegt haben und die Tags dann in die Datenbasis hochgeladen haben.

Wie kann ich selbst Musik hinzufügen oder Tags festlegen?

Dafür braucht man ein Programm (weil die Musik identifiziert werden muss, siehe Technik). Derzeit geht es mit dem Musik- und Soundplayer Ares. Dort wählt man für Musik (die man bereits auf dem Rechner hat) Tags aus und lädt sie dann hoch.

Kann ich auch selbst ein Programm schreiben, das die Daten nutzt?

Ja, siehe dazu die Beschreibung der API.
